# Catnip's WCL Mage Analyzer
(for WoW Classic MoP)

This simple web app analyzes your Warcraftlogs Log and gives suggestions on how to improve your gameplay. Examples are:
- Hit cap
- Missing Enchants
- Bomb uptime and proper clipping
- Evocation/Rune of Power uptime
- Gloves usage
- ...

Visit https://catnipcode.github.io/mage-log-analyzer/ to use the web app.
